# Tools

Small helper scripts used during cleanup and CI hardening.

- `fix_all_missing_docs.sh`: helps address missing rustdoc warnings.
- `fix_clippy_5_cat.sh`: runs targeted clippy fixes in batches.
- `fix_clippy_allow_all.sh`: temporary allowlist-based clippy pass (use sparingly).
- `dedup_inner_attrs.sh`: removes duplicated inner attributes.
